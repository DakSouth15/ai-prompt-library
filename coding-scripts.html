⚡ Automation Scripts

Automate the boring stuff
in minutes

Prompts for Python, JavaScript, and bash scripts that save you hours of repetitive manual work.

01
Write a Python script that watches a folder for new files and automatically renames them based on a pattern. The pattern is: [describe your naming convention — e.g. date-prefix, lowercase, remove spaces]. Run continuously in the background, log each rename to a text file, and skip files that already match the pattern.
02
Create a Python script that reads a CSV file at [path], processes each row by [describe what to do — e.g. format dates, clean phone numbers, remove duplicates], and writes the cleaned output to a new CSV. Print a summary showing how many rows were processed, changed, and skipped.
03
Write a bash script that backs up a directory to a timestamped folder automatically. It should: create a backup at ~/backups/YYYY-MM-DD/, compress it as a .tar.gz, delete backups older than 30 days, log success or failure to a log file, and be set up to run daily via cron. Show me the cron command too.
04
Build a Python web scraper that visits [URL], extracts [what data — e.g. product names, prices, links], and saves it to a CSV file. Handle pagination by following "next page" links until there are none. Add a 1-2 second delay between requests, respect robots.txt, and include error handling for failed requests.
05
Write a JavaScript/Node.js script that reads all markdown files in a folder, extracts the H1 title and first paragraph from each, and generates a JSON index file with each file's name, title, excerpt, and word count. This will be used to build a search index for a static site.
06
Create a Python script that sends me a daily email summary using the Gmail API. It should pull data from [source — e.g. a CSV, an API, a folder], format it as a clean HTML email with a subject line of "Daily Summary - [date]", and send it to [email address] every morning at 8am. Show me the full setup including OAuth.
07
Build a Python script that monitors a website for changes. Check [URL] every [X] minutes, compare the current page content to the last saved version, and send me a desktop notification or email if anything has changed. Save a timestamped snapshot each time a change is detected. Make it run as a background process.
08
Write a script that automates resizing and optimizing images in a folder. For every image in [input folder]: resize it to max width [X]px while keeping aspect ratio, compress it to under [X]KB, convert it to WebP format, and save it to [output folder] with the same filename. Use Python with Pillow. Print the before/after file sizes for each image.
